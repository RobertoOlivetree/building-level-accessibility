# Building-Level-Accessibility

## Reproducible Workflow for Building-Level Accessibility Analysis

This repository contains the complete reproducible workflow accompanying the manuscript:

**Building-Level Accessibility for Older Adults in the 15-Minute City: Effects of Spatial Aggregation and Slope-Adjusted Pedestrian Networks**

The repository includes all Python notebooks, input datasets, intermediate processing steps, and analytical outputs required to reproduce the results presented in the manuscript.

## Workflow

### Notebook 01 – Building Accessibility

Computes building-level accessibility independently for each parish using the pedestrian network and the predefined accessibility parameters.

**Output**

- One accessibility dataset for each parish.

### Notebook 02 – Merge Parishes

Combines the outputs from the seven parish-level analyses into a single study-area dataset.

**Output**

- `resultados_unidos.csv`

### Notebook 03 – Final Analysis

Reproduces all analyses reported in the manuscript, including:

- Building-level accessibility analysis
- Population integration
- Principal Component Analysis (PCA)
- Urban Attractiveness Index
- IVEA
- Global Moran's I
- Local Indicators of Spatial Association (LISA)
- Spatial aggregation analysis
- Tables and figures

## Reproducibility

Execute the notebooks in the following order:

1. `01_building_accessibility.ipynb`
2. `02_merge_parishes.ipynb`
3. `03_final_analysis.ipynb`

Running the notebooks sequentially reproduces all analyses, tables and figures reported in the manuscript.
